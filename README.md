# Library Management System

## Overview
A Library Management System developed as a university group project using PHP and MySQL. The system helps manage books, student records, and book issuing operations.

## Features
- User Login
- Admin Dashboard
- Book Management
- Issue & Return Books
- Student Management
- Password Recovery

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## My Contribution
This project was developed as part of a university group project. I contributed as a team member by participating in development, testing, debugging, and implementing features.

## How to Run
1. Install XAMPP.
2. Place the project folder inside the `htdocs` directory.
3. Import `admin.sql` into phpMyAdmin.
4. Start Apache and MySQL.
5. Open the project in your browser.
