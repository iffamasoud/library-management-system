# Library Management System

A web-based Library Management System developed as a university group project.

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- Bootstrap
- JavaScript

## Features
- Student Registration
- Student Login
- Admin Login
- CAPTCHA Verification
- Book Issue & Return Management
- User Dashboard
- Password Change
- Profile Management
- Email Availability Check

## Database
Import `admin.sql` into MySQL before running the project.

## Note
This project was completed as part of a university group assignment. I contributed as one of the team members.
